¿Qué permite hacer MCP?

Una vez conectado, Claude puede leer, consultar y escribir en las colecciones de Datum V2 directamente en lenguaje natural, sin necesidad de escribir código SQL.

  • Datum es el gestor de bases de datos clásico de Jelou. Datum V2 es una plataforma dedicada (datum.jelou.ai) con API REST nativa, monitoreo de recursos, triggers y compatibilidad MCP para IA.
  • Una acción automática que se ejecuta cuando ocurre un evento en la base de datos (insertar, actualizar o eliminar un registro). Se pueden conectar a URLs externas (webhooks) para notificar a otros sistemas.

Glosario

Términos Clave

Definiciones rápidas de los conceptos más usados en Jelou.AI

Agente
Entidad de IA en Brain Studio que entiende y responde con lógica personalizable.
Canvas
Área visual de construcción de flujos en Brain Studio. Superficie de trabajo drag-and-drop.
Skill
Módulo reutilizable con lógica específica. Se puede usar en múltiples proyectos.
Tool
Integración con un servicio externo (API, CRM, pasarela de pago, etc.).
MCP
Model Context Protocol — conecta bases de datos directamente con herramientas de IA como Claude o ChatGPT.
Trigger
Evento que activa una acción automática en Datum V2 (insertar, actualizar o eliminar registro).
Traza
Secuencia detallada de pasos ejecutados en un flujo, visible en el detalle de un Log.
SKU
Stock Keeping Unit — código único que identifica un producto en el catálogo de Shop.
HSM
Highly Structured Message — plantilla de mensaje aprobada para envíos masivos por WhatsApp.
Colección
Tabla de datos dentro de Datum V2. Equivalente a una tabla en una base de datos relacional.
Debugger
Herramienta de Brain Studio para inspeccionar conversaciones reales de usuarios finales.
Human Attention
Nodo de Brain Studio que transfiere la conversación a un agente humano en Connect.
